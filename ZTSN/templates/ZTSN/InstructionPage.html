Überblick:

Die folgende Marktsimulation wird mithilfe einer browserbasierten Handelsoberfläche dargestellt. Die Aufgabe besteht aus 3 Runden: 1 Übungsrunde und 2 Testrunden. Sobald alle Runden absolviert wurden, werden Sie eine Auszahlung in Abhängigkeit von Ihren Ergebnissen bekommen, wobei nur eine der beiden Testrunden zufällig ausgewählt wird.

Regeln Marktsimulation:
Zu Beginn jeder Runde erhalten Sie ein Portfolio, bestehend aus einem sicheren Vermögenswert (Bargeld) im Wert von 120 ECUs, wobei Sie hiervon 80 ECUs am Ende wieder abgeben müssen, und 0 Anteile von einem riskanten Vermögenswert (Aktie). In jeder Runde können Sie Aktien kaufen und verkaufen indem Sie auf die "Kaufen" oder "Verkaufen" Schaltflächen klicken. Das können Sie so oft machen wie Sie glauben, dass es für Sie gewinnbringend sein wird. Behalten Sie aber stets in Erinnerung, dass Sie eine Gebühr in Höhe von 1,25% (in Relation zum aktuellen Aktienpreis) für jeden Kauf und Verkauf zahlen. Sie können immer nur 1 Aktie oder 5 Aktien kaufen und 1 Aktie oder 5 Aktien verkaufen. Besitzen Sie nur 3 Aktien, können Sie mit der "5x verkaufen" Schaltfläche Ihre maximale Zahl an Aktien verkaufen. Nehmen Sie nun beispielsweise an, dass Sie eine Aktie halten und diese für einen Marktpreis von 100 ECUs verkaufen, dann werden Ihnen nur 98,75 ECUs auf Ihr Bargeldkonto gutgeschrieben. Sind Sie sich außerdem bewusst, dass der Handel für Sie für 2 Sekunden ausgesetzt wird, wenn Sie eine Aktie verkaufen wollen, aber keine besitzen oder wenn Sie eine Aktie kaufen wollen und nicht genug Bargeld für den Marktpreis und zusätzlich für die Kaufgebühr in Höhe von 1,25% haben. Jeder Handel, Kauf oder Verkauf, wird unverzüglich zum aktuellen Marktpreis durchgeführt. Die Marktpreise basieren auf historischen Kursen von realen Aktien. Aktien sind durch eine höhere Volatilität gekennzeichnet als Bargeld. Leerverkäufe sind nicht möglich.

Auszahlung:
Sie erhalten eine leistungsabhängige Auszahlung aus einer zufällig ausgewählten Runde der beiden Testrunden. Die Auszahlung entspricht dem Gesamtwert des Portfolios am Ende der Runde (nach Rückzahlung der 80 ECUs). Verluste werden nicht von Ihrer fixen Vergütung abgezogen.
